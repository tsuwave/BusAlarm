# TsuWave

## Secrets

Do not commit `Secrets.xcconfig` to version control. It contains your private API key.

The file is already added to `.gitignore`.

## Getting an LTA DataMall API Key

1. Visit https://datamall.lta.gov.sg/content/datamall/en/request-for-api.html
2. Register for an account
3. Request API access
4. Copy your Account Key to `Config/Secrets.xcconfig`
